Covid-19 Cases Analysis for US, UK and India

This is a narrative visualization implementing by the D3 library in JavaScript and the interactive slide show of one effective narrative visualization structures. The dataset is using latest (7/14/2021) Covid-19 Dataset provided by OurWorldInData.Org. Right now, people are worried about the Delta variant of Covid-19 spreading. This is project to find the spreading pattern of the situation of the Covid-19 cases in the United States, United Kingdom, and India.

Jiayu Peng
July 16, 2021

Messaging

I use narrative visualization to better present the data changed among those three countries' covid-19 cases comparison. The narrative visualization helps researchers to find out the pattern of the virus spreading and the spreading speed among these countries. The general idea that I present is the correlation between the timeline and the 7-day average cases from United States, United Kingdom and India, from Jan, 2020 to Jul, 2021.

Narrative Structure

I choose an interactive slideshow to present this data visualization. On the navigation bar, click the Visualization tab, then it directs to my visualization page.
Under the header, there are three tabs and each tab can show a scene of the country-to-country comparison in Covid-19 cases. Each scene focuses on the different topics of the story. Besides, I made the feature that users can freely pick up the country name to see the data from that country. That gives users the opportunity to explore the scene. Moreover, I add the features such as tooltip and interactive legend.

Visual Structure

Firstly, I use connected scatterplot as my scenes to present the Covid-19 cases data. Secondly, for each line on the scatterplot is represented a country's data, and I add a feature that can hide or show the line. That can help the users to select their interested country data. Thirdly, I use a dynamic visualization in annotations and scatterplot chart, which much improved the user experience in explore the story.

Scenes

There are three scenes. The first scene is the cases comparison between United States and United Kingdom, which can reflected the difference in cases number and peak value between US and UK. We can see the obvious difference that US had much higher peak than UK. However, we can see the peak date is very closed for the two countries. The second scene is about the comparison between United States and India, because this scene can better show the Delta variant impact on India, even US got higher peak than UK, however, the India peak was 1.5 times than US. That can prove that the Delta variant has extremely infectious. The last scene is about the after pandemic, three countries' comparison, which show the vaccination great impact.

Annotations

I use the annotation class to manage all the annotations in different scenes in the stepper.css file, I design the color, margin, font-size to modify those annotations in the similar style, that can be considered as the "format". I think the advantage of the format is better to read and easy to recognize without any confuse. Yes, the annotation is used to support the messaging, the annotations points out the feature of each scene and help users to figure out the story path. The annotation can be changed in different steppers. Because each stepper has different topics.

Parameters

There are several parameters in the narrative visualization, they are the country parameter and the date parameter. The 7-day average variable is the states and controlled by the country parameter and date parameter.
Firstly, the scatterplot is presented by the country parameter and the date parameter. Secondly, when the scene parameter changed, country parameter will make the related the states of cases changed simultaneously. Thirdly, there is interactive legend feature. when the country name filter is selected, the states value can be hidden or shown up in the scene.

Triggers

For the triggers that connect user actions to changes of state in the narrative visualization, there are 3 different ways: the stepper of each scene, the interactive legend and the tooltip.
The user can choose either to hover the cursor to the any datapoint, or changed the scene parameter to get the exploration for the story, or use interactive legend to choose any country interested in.
